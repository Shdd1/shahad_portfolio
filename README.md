# Shahad Code

Personal portfolio website for Shahad, a Software Engineer focused on backend systems, REST APIs, and clean digital experiences.

## Live Project

This repository contains a static portfolio site built with plain HTML and CSS. It is ready to deploy on GitHub Pages, Netlify, Vercel, or any static hosting provider.

## Features

- Responsive portfolio layout
- Light theme with purple and cyan accents
- Custom hero illustration
- About, skills, projects, and contact sections
- Static assets included in the `assets` folder

## Tech Stack

- HTML5
- CSS3
- Remix Icon
- Devicon

## Project Structure

## Run Locally

Open `index.html` directly in a browser, or run a simple local server:

```bash
python -m http.server 3000
```

Then visit:

```text
http://localhost:3000
```

## Domain

The intended domain for this project is:
